### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iion91/mandelbrot.git
   ```
   or download the file
2. Install dependencies
   ```sh
   pip install numba
   pip install numpy
   pip install matplotlib
   ```
3. Start the script 
   Enter your the path and type
   ```cmd
   python mandelbrot.py
   ```
